# Punnerruslaski 💪

PWA-sovellus päivittäisten punnerrusten seurantaan.

## Ominaisuudet

- 📊 Tilastot viikko / kuukausi / vuosi -näkymissä
- 📋 Historia ja merkintöjen muokkaus
- 📱 Toimii offline (PWA)
- 💾 Tiedot tallennetaan laitteelle (localStorage)

## Asennus iPhoneen / iPadiin

1. Avaa sivu Safarissa: `https://vdy8qdbddj-creator.github.io/Punnerrus/`
2. Paina **Jaa**-painiketta (neliö nuolella)
3. Valitse **Lisää Koti-valikkoon**
4. Paina **Lisää** – sovellus ilmestyy kotinäytölle

## GitHub Pages -julkaisu

1. Luo uusi repository nimellä `punnerruslaski`
2. Kopioi kaikki tiedostot repositoryyn
3. Mene **Settings → Pages**
4. Valitse Source: **Deploy from a branch → main → / (root)**
5. Tallenna – sivu julkaistaan minuuteissa
